# 🚀 AI 시그널 퀄리티 업그레이드 및 인프라 최적화

어젯밤부터 오늘까지 진행된 핵심 작업은 **AI 요약 결과의 품질을 최고 수준으로 끌어올리는 것**과 **API 할당량 초과(Rate Limit) 문제를 근본적으로 해결**하는 두 가지 축으로 진행되었습니다. 더불어 사용자 편의성을 위한 실시간 자동 새로고침 기능과 프리미엄 리드미(README) 업데이트도 완료했습니다.

## 1. 🤖 AI 프롬프트 엔지니어링 및 요약 품질 개선 (Prompt Refinement)

기존 시그널 요약에서 몇 가지 아쉬운 점이 발견되었습니다. `short_reason`에 쉼표(,)가 너무 자주 사용되거나 구/절 명사형태가 아닌 단어 나열식인 경우, 그리고 번역투의 뻔한 기계적 문장이 등장하는 현상입니다.

- **명사구/어절 단위 강제화**: `short_reason` 작성 시 쉼표 사용을 전면 금지하고, 문맥이 이어지는 2~4어절의 단일 구/절 단위로 작성하도록 프롬프트를 전면 개편했습니다. (예: "외국인 대규모 매수세로 상승")
- **기계적 서술어 금지**: "~등의 영향으로 상승 마감했습니다", "~로 인해 주가가 상승했습니다" 등 뻔한 문장 구조를 AI가 절대 사용하지 못하도록 네거티브 프롬프트(Negative Prompt)를 강력하게 적용했습니다.
- **다국어(US) 처리 강화**: 미국 주식 뉴스를 영어 그대로 주입하되, 어떤 상황에서도 반드시 한국인 독자를 위해 **한국어로 요약 및 이유를 반환**하도록 명시적 지시를 추가했습니다.

## 2. 📉 지수(Index) 요약 로직의 지능화

우연히 발견한 이슈로, 코스피나 나스닥 같은 시장 지수에 대한 요약 프롬프트가 '개별 기업'용 프롬프트를 그대로 타고 들어가면서 문맥이 어색해지는 현상이 있었습니다.

- **실제 등락률(change_rate) 주입**: 기존에는 지수 요약에 변동률을 0.0으로 고정해서 넘겼으나, 파싱 로직을 개선해 실제 지수의 등락률을 Gemini에게 넘기도록 수정했습니다.
- **Index vs Stock 콘텍스트 분리**: `KOSPI`, `S&P 500` 등의 이름이 감지되면 프롬프트가 개별 종목이 아닌 **"시장 지수"** 변동을 분석하도록 동적 변수(`is_index_prompt`)를 적용해 지수 요약의 자연스러움을 대폭 끌어올렸습니다.

## 3. ⚡ Smart Skip 및 --force 강제 재생성 로직

Gemini API의 할당량(Quota) 초과 에러(429)가 빈번하게 발생하는 문제를 해결하기 위해 **Smart Skip** 인프라를 도입했습니다.

- **중복 호출 방지**: 어제와 동일한 기사/상태이고 이미 성공한 AI 요약(`summary_success: true`)이 존재한다면, 불필요하게 API를 호출하지 않고 기존의 요약을 캐싱하여 재사용합니다.
- **--force 플래그 도입**: 로컬에서 개발 목적으로 최신 프롬프트를 적용하여 기존 데일리를 완전히 새로 덮어쓰고 싶을 때 사용하는 `--force` 시스템을 구축했습니다.
  ```bash
  python backend/crawler.py --date 2026-03-06 --market US --force 
  ```
  이 기능을 활용해 기존에 생성되었던 03월 06일자 퀄리티 낮은 데이터를 전부 고품질 AI 요약본으로 교체 성공했습니다.

## 4. 🔄 Streamlit 실시간 자동 새로고침(Auto-Refresh) 적용

GitHub Actions가 매 20분마다 새로운 데이터를 수집해 레포지토리에 푸시하더라도, 현재 웹페이지를 켜둔 접속자는 F5를 누르지 않으면 구형 데이터를 보고 있어야 했습니다.

- **해결책**: `streamlit-autorefresh` 패키지를 도입했습니다. 날짜 선택란 옆에 **[🔄 5분 자동 새로고침]** 체크박스를 추가하여, 이 체크박스를 활성화한 사용자의 화면은 스스로 5분마다 최신 데이터를 가져오도록 UX를 대폭 개선했습니다.

## 5. 🎨 프리미엄 README.md 전면 개편

GitHub 레포지토리의 얼굴인 `README.md` 문서를 토스(Toss)와 같은 모던하고 프리미엄 IT 기업의 톤앤매너로 완전히 새롭게 작성했습니다. 프로젝트의 아키텍처, 핵심 AI 기능, 자동화 인프라 파이프라인이 매력적으로 돋보일 수 있도록 배지를 달고 상세 설명을 보강했습니다.
